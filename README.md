# Роб Бот — лендинг

Статическая посадочная страница продукта «Роб Бот» (AI-агент отдела продаж): hero, боли, социальное доказательство, шаги, Human-in-the-Loop, RAG/CRM, сравнение, тарифы, FAQ и финальный CTA.

## Стек

- Чистый **HTML + CSS + JS**, без сборки и без Tailwind.
- **Дизайн-токены** (цвета, типографика, отступы, радиусы, тени, motion) задаются в `:root` в [`styles.css`](styles.css) — это канонические значения; [`design-system.md`](design-system.md) описывает арт-направление.
- Шрифт: **Plus Jakarta Sans** (Google Fonts). Иконки: **Font Awesome 6** (CDN).

## Структура проекта

| Файл | Назначение |
|------|------------|
| `index.html` | Семантическая разметка и контент |
| `styles.css` | Токены, компоненты, адаптив, reveal, `prefers-reduced-motion` |
| `main.js` | Sticky header (`is-scrolled`), мобильное меню, `IntersectionObserver` для `.reveal` |

## Как открыть локально

Нужен локальный HTTP-сервер (корректная загрузка `main.js` и `styles.css`).

```bash
cd "/path/to/robbot"
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.
