# Роб Бот — лендинг

Статическая посадочная страница продукта «Роб Бот» (AI-агент отдела продаж): hero, блоки болей, социальное доказательство, как работает, Human-in-the-Loop, RAG/CRM, сравнение, тарифы, FAQ и финальный CTA.

## Структура проекта

| Файл | Назначение |
|------|------------|
| `index.html` | Разметка страницы |
| `styles.css` | Кастомные анимации, утилиты (градиентный текст, reveal, reduced motion) |
| `main.js` | Конфигурация темы Tailwind (цвета `brand` / `accent`, шрифт Inter) и `IntersectionObserver` для появления блоков `.reveal` |

Внешние зависимости подключаются из CDN: [Tailwind CSS](https://tailwindcss.com/docs/installation/play-cdn), Google Fonts (Inter), Font Awesome 6.

## Как открыть локально

Нужен локальный HTTP-сервер (из-за загрузки `main.js` и `styles.css` по относительным путям; `file://` может блокировать модули в некоторых браузерах).

Пример с Python:

```bash
cd "/path/to/robbot"
python3 -m http.server 8080
```

Затем откройте в браузере: `http://localhost:8080`.

## Заметки

- Якорь `#guide` в начале `<body>` зарезервирован под ссылку «Скачать гайд»; при появлении PDF замените `href` на файл или внешний URL.
- Якорь `#demo` ведёт на финальный блок с призывом к демо.
